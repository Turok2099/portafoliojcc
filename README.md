# Jorge Castro — Portfolio

Landing page personal: Full Stack Developer & SEO Specialist. Proyectos en producción (Triple Impacto, The Klan BJJ, La Trinchera México).

## Stack

- **Next.js 15** · **React 19** · **TypeScript**
- **Tailwind CSS** · **Framer Motion** · **react-icons** · **react-intersection-observer**

## Scripts

```bash
npm install
npm run dev    # Desarrollo
npm run build  # Producción
npm run start  # Servidor producción
```

## Deploy en Vercel

1. Conectar el repositorio en [Vercel](https://vercel.com).
2. Deploy automático con `npm run build` y `npm run start`.

No se requieren variables de entorno para el deploy básico. Las imágenes usan Cloudinary (URLs externas).
